# 🤖 AUTO-REPLY-AI-CHATBOT

> 💬 This project automates the process of interacting with a chat application, specifically designed to analyze chat history and generate humorous responses using OpenAI's GPT-5-nano model.

---

## 📌 Overview

This virtual assistant, named **Ubaid**, is a character that:

- 😄 Roasts people in a funny way  
- 🧠 Analyzes chat history  
- 🤖 Generates smart and humorous replies  

---

## ⚙️ How It Works

- 📥 Reads chat history  
- 🧠 Processes conversation context  
- 😂 Generates humorous responses  
- 💬 Sends automated replies  

---

## 🧠 AI Model Used

- OpenAI **GPT-5-nano**

---

## 🎯 Purpose

To create an intelligent chatbot that:
- Automates conversations  
- Adds humor to chats  
- Enhances interaction experience  

---

## 🚀 Summary

A fun and smart AI chatbot that:
- Understands conversations  
- Generates witty replies  
- Automates chat interactions
